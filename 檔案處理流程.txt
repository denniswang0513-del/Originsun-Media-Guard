│
├─ 否 ──────────────────────────────────────────→ 直接複製
│
└─ 是
   │
   ① 檔案大小相同？
   │
   ├─ 否（大小不同）─── 疑似損壞的舊備份
   │   對話框：[覆蓋此檔] [全部覆蓋] [略過此檔] [全部略過] [更改檔名]
   │   ├─ 覆蓋      → 複製（覆蓋舊檔）
   │   ├─ 略過      → 跳過此檔
   │   └─ 更改檔名  → 複製為 檔名(1).ext, (2)...
   │
   └─ 是（大小相同）
      │
      ② 修改時間差異 ≤ 2 秒？
      │
      ├─ 否（時間不同）─── 可能是不同版本
      │   對話框：[覆蓋此檔] [全部覆蓋] [略過此檔] [全部略過] [更改檔名] [XXH64 校驗]
      │   ├─ 覆蓋      → 複製
      │   ├─ 略過      → 跳過
      │   ├─ 更改檔名  → 複製為 檔名(1).ext...
      │   └─ XXH64 校驗 → 計算 Hash
      │       ├─ Hash 相同 → 安靜略過（完全相同）
      │       └─ Hash 不同 → 對話框：[覆蓋目標] [略過此檔] [更改檔名]
      │
      └─ 是（時間在容差內）
         │
         ③ 依 XXH64 勾選狀態分流
         │
         ├─ 已勾選 XXH64 → 自動計算 Hash
         │   ├─ Hash 相同 → 安靜略過
         │   └─ Hash 不同 → 靜默損壞！
         │       對話框：[覆蓋目標] [略過此檔] [更改檔名]
         │
         └─ 未勾選 XXH64 → 對話框：[安靜略過] [XXH64 校驗]
             ├─ 安靜略過  → 跳過（信任 Metadata）
             └─ XXH64 校驗 → 計算 Hash
                 ├─ Hash 相同 → 安靜略過
                 └─ Hash 不同 → 靜默損壞！
